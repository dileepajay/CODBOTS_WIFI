ii_WIFI.h Basic Example
This page enables direct interaction with WiFi-connected LEDs. Utilize the LED ON/OFF buttons for immediate effect and access WiFi settings for device configuration.




Note:
  • Click the LED ON button to send a request to the server at /ledon, turning the LED on.
  • Click the LED OFF button to send a request to the server at /ledoff, turning the LED off.
  • Click the Settings button to navigate to wifi_connect.html for WiFi configuration options.
